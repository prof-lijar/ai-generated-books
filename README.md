# AI-Generated Books

Books written overnight by AI agents using locally-running LLMs.

Each book is generated by the [book-writer agent](https://github.com/prof-lijar/orchast_agent/tree/master/book-writer), which runs a 4-phase pipeline per chapter (outline, write, review, finalize) using Gemma models via Ollama.

## How It Works

1. Create a table of contents (JSON/YAML)
2. Run the book-writer agent overnight
3. Wake up to a finished book, committed chapter by chapter

```bash
cd ../orchast_agent/book-writer
python run_book.py --toc /path/to/toc.json --output-dir /path/to/book-dir --repo https://github.com/prof-lijar/ai-generated-books.git
```
