# Chapter 2: HTML အခြေခံ — ဝဘ်စာမျက်နှာရဲ့ အရိုးစု (The Skeleton of a Webpage)

သင် ဘယ်လို Website တွေ မြင်ရလဲ? အဲဒီ Website တွေရဲ့ နောက်ကွယ်မှာ ဘာတွေ ရှိနေသလဲဆိုတာကို သိချင်ပါသလား။ အင်တာနက်ပေါ်ရှိ အရာအားလုံးဟာ မြင်ရတာထက် ပိုမိုရှုပ်ထွေးတဲ့ ကွန်ရက်တစ်ခုအပေါ် အခြေခံနေပါတယ်။ ဒီအခန်းမှာ ကျွန်ုပ်တို့သည် အဆိုပါ အခြေခံကို အဆင့်ဆင့် နားလည်ပြီး၊ ဝဘ်စာမျက်နှာတစ်ခုကို တည်ဆောက်ရာတွင် အခြေခံကျသော ဘာသာစကားဖြစ်သည့် HTML (HyperText Markup Language) ကို မည်သို့အသုံးပြုရမည်ကို လေ့လာကြပါမည်။ HTML ဆိုသည်မှာ ဝဘ်စာမျက်နှာ၏ အခြေခံ အရိုးစု (Skeleton) ကဲ့သို့ပင် ဖြစ်သည်။ ဤအရိုးစုကို နားလည်မှသာ နောက်ပိုင်းတွင် CSS ဖြင့် အလှအပနှင့် JavaScript ဖြင့် အသက်ဝင်မှုကို ထည့်သွင်းနိုင်မည် ဖြစ်ပါသည်။

## 1. Introduction: Webpage ရဲ့ အခြေခံကို နားလည်ခြင်း (Understanding the Basics of a Webpage)

ကျွန်ုပ်တို့သည် အင်တာနက်ပေါ်တွင် တစ်စုံတစ်ရာကို ကြည့်ရှုသောအခါ၊ ကျွန်ုပ်တို့ မြင်တွေ့ရသည်မှာ စာသားများ၊ ပုံများ၊ ဗီဒီယိုများ၊ လင့်ခ်များ စသည်တို့ ဖြစ်ကြသည်။ ဤအရာအားလုံးသည် ကွန်ပျူတာဆိုက်များမှ အချက်အလက်များကို စုစည်းတင်ပြထားခြင်း ဖြစ်သည်။ ထိုအချက်အလက်များကို ဖော်ပြရန်အတွက် အသုံးပြုသော မူလဘာသာစကားမှာ HTML ဖြစ်သည်။

### HTML ဆိုတာ ဘာလဲ? (What is HTML?)

HTML ဆိုသည်မှာ Markup Language သဘောတရားကို အခြေခံထားသော ဘာသာစကားတစ်ခု ဖြစ်သည်။ Markup Language ဆိုသည်မှာ အချက်အလက်များကို စီစဉ်ပုံ (Structure) နှင့် ဖော်ပြပုံ (Presentation) ကို သတ်မှတ်ပေးသည့် စည်းမျဉ်းများ ဖြစ်သည်။ အချက်အလက်များကို တိုက်ရိုက်ဖော်ပြမည့်အစား၊ အချက်အလက်အမျိုးမျိုးကို အထပ်ထပ် စီစဉ်ပေးခြင်းဖြင့် ကွန်ပျူတာနှင့် Browser များအနေဖြင့် ထိုအချက်အလက်များကို မှန်ကန်စွာ နားလည်ပြီး ပြသနိုင်စေသည်။ HTML ၏ အဓိက ရည်ရွယ်ချက်မှာ ဝဘ်စာမျက်နှာပေါ်ရှိ အကြောင်းအရာများကို စနစ်တကျ စီစဉ်ပြီး အင်တာနက်ပေါ်တွင် ထုတ်လွှင့်နိုင်ရန် ဖြစ်သည်။

### Webpage ရဲ့ အစိတ်အပိုင်းများ (Components of a Webpage)

HTML ကို နားလည်ရန်အတွက် အခြေခံဝေါဟာရများကို သိရှိရန် လိုအပ်ပါသည်။ ထိုဝေါဟာရများမှာ Element, Tag, နှင့် Attribute တို့ ဖြစ်ကြသည်။

**Element:** HTML ၏ အခြေခံ အစိတ်အပိုင်းတစ်ခုဖြစ်သည်။ ၎င်းသည် စာမျက်နှာပေါ်ရှိ အကြောင်းအရာတစ်ခုလုံးကို ကိုယ်စားပြုသည်။ ဥပမာအားဖြင့်၊ စာပိုဒ်တစ်ခုလုံးသည် Element တစ်ခု ဖြစ်သည်။

**Tag:** Element များကို အဖွင့် (Opening) နှင့် ပိတ် (Closing) အမှတ်အသားများဖြင့် ဖော်ပြခြင်းကို Tag ဟုခေါ်သည်။ Tag များသည် HTML အတွက် အဓိပ္ပာယ်ပေးသည်။ ဥပမာအားဖြင့်၊ စာပိုဒ်တစ်ခုကို ဖော်ပြရန် `<p>` နှင့် `</p>` ကို အသုံးပြုသည်။

**Attribute:** Element များ၏ ဂုဏ်သတင်း (Properties) သို့မဟုတ် အပိုဆောင်း အချက်အလက်များကို ပေးရန် အသုံးပြုသည်။ Attribute များသည် Element ကို ပိုမို အဓိပ္ပာယ်ရှိအောင် ပြုလုပ်ပေးသည်။ ဥပမာအားဖြင့်၊ ပုံတစ်ခုကို ထည့်သွင်းရာတွင် `src` (Source) attribute ကို အသုံးပြုသည်။

ဤသုံးမျိုးကို ဥပမာဖြင့် ရှင်းပြကြည့်ရအောင်။ စာသားတစ်ခုကို ဖော်ပြလိုလျှင် ကျွန်ုပ်တို့သည် `<p>` နှင့် `</p>` ကို အသုံးပြုသည်။ ဤ `<p>` နှင့် `</p>` သည် စာပိုဒ်တစ်ခုလုံးကို ပတ်လည်ထားသည်။ ထို့နောက် ထိုစာပိုဒ်၏ အကြောင်းအရာကို ပိုမို အထူးပြုရန်အတွက် `<h1>` (Heading) ကို အသုံးပြုသည်။ ဤနေရာတွင် `<h1>` သည် Element ဖြစ်ပြီး၊ ၎င်းသည် စာမျက်နှာ၏ အရေးကြီးဆုံး ခေါင်းစဉ်ကို ဖော်ပြသည်။ ထို့အပြင် ထိုခေါင်းစဉ်ကို ဘရောက်ဆာတွင် အထူးပုံစံဖြင့် ပြသစေရန်အတွက် `<h1>` ၏ Attribute အဖြစ် `class="title"` ဟု ထည့်သွင်းနိုင်သည်။ ဤနည်းအားဖြင့် HTML သည် အချက်အလက်များကို စနစ်တကျ စီစဉ်ပေးနိုင်သော စနစ်တစ်ခု ဖြစ်လာသည်။

**Transition Note:** HTML ဟာ ဝဘ်စာမျက်နှာရဲ့ အခြေခံ အရိုးစု (Skeleton) လိုပဲ။ ဤအရိုးစုကို နားလည်မှသာ CSS နဲ့ JavaScript တို့နှင့် ပေါင်းစပ်ပြီး အလှအပနှင့် အသက်ဝင်မှုကို ထည့်နိုင်မှာ ဖြစ်သည်။

## 2. HTML Structure: စာမျက်နှာရဲ့ အခြေခံဖွဲ့စည်းပုံ (The Basic Structure of an HTML Page)

မည်သည့် ဝဘ်စာမျက်နှာမဆို အခြေခံကျသော ဖွဲ့စည်းပုံတစ်ခု ရှိသည်။ HTML သည် ထိုဖွဲ့စည်းပုံကို မည်သို့တည်ဆောက်ရမည်ကို သတ်မှတ်ပေးသည်။

### Document Type နှင့် Root Element

HTML စာမျက်နှာတစ်ခု စတင်ရန်အတွက် အရေးကြီးဆုံး အချက်မှာ Document Type ကို သတ်မှတ်ပေးခြင်း ဖြစ်သည်။ ကျွန်ုပ်တို့သည် စာမျက်နှာသည် HTML 3.2 သို့မဟုတ် နောက်ဆုံးပေါ် ဗားရှင်းကို အသုံးပြုကြောင်း ဖော်ပြရန်အတွက် `<!DOCTYPE html>` ဟူသော စာသားကို စာမျက်နှာ၏ အစတွင် ရေးသားရမည် ဖြစ်သည်။ ဤစာသားသည် Browser တစ်ခုအား ဤစာမျက်နှာသည် HTML5 စံနှုန်းအတိုင်း ရေးသားထားကြောင်း ညွှန်ကြားပေးသည်။

ထို့နောက် အဆိုပါ HTML စာမျက်နှာ၏ အစအဆုံးကို ဝန်းရံပေးသော အကြီးမားဆုံး Element ဖြစ်သည့် `<html`> Tag ကို အသုံးပြုရမည်။ ဤ `<html`> Tag သည် စာမျက်နှာ၏ အစအဆုံး အကြောင်းအရာအားလုံးကို စုစည်းထားသော အဓိက အမြစ် (Root Element) ဖြစ်သည်။

### Head Section (ခေါင်းပိုင်း)

`<html`> Tag အတွင်းတွင် အကြောင်းအရာများကို ခွဲခြားဖော်ပြရန်အတွက် ကျွန်ုပ်တို့သည် အပိုင်းနှစ်ပိုင်း ခွဲ၍ ရေးသားရမည် ဖြစ်သည်။ ပထမအပိုင်းမှာ `<head>` Tag ဖြစ်သည်။ `<head>` Tag သည် စာမျက်နှာ၏ ခေါင်းပိုင်း (Head Section) ကို ကိုယ်စားပြုပြီး၊ ၎င်းသည် စာမျက်နှာပေါ်တွင် တိုက်ရိုက်မပြသသော်လည်း အရေးကြီးသော မက်တာအချက်အလက်များ (Meta Information) ကို ထည့်သွင်းရန် အသုံးပြုသည်။ ဥပမာအားဖြင့်၊ စာမျက်နှာ၏ ခေါင်းစဉ်ကို သတ်မှတ်ရန်အတွက် `<title>` Tag ကို `<head>` အတွင်း၌ ထည့်သွင်းရသည်။ ဤ `<title>` Tag ၏ အကြောင်းအရာသည် Browser ၏ Tab Bar တွင် ပေါ်လာမည့် ခေါင်းစဉ်အဖြစ် အရေးကြီးပါသည်။ ထို့အပြင် နောက်ပိုင်းတွင် CSS ဖိုင်များကို ချိတ်ဆက်ရန်အတွက်လည်း ဤနေရာတွင် Link လုပ်ရန်အတွက် အသုံးပြုသည်။

### Body Section (ကိုယ်ထည်ပိုင်း)

`<head>` အပိုင်းပြီးနောက်တွင် လာသော အပိုင်းမှာ `<body`> Tag ဖြစ်သည်။ `<body`> Tag သည် ဝဘ်စာမျက်နှာ၏ အဓိက အကြောင်းအရာများ (Visible Content) အားလုံးကို ကိုယ်စားပြုသည်။ စာမျက်နှာပေါ်တွင် အသုံးပြုသူများ မြင်တွေ့ရမည့် စာသားများ၊ ပုံများ၊ ဗီဒီယိုများ၊ စာရင်းများ စသည်တို့သည် အားလုံးကို `<body`> Tag အတွင်း၌ ရေးသားရမည် ဖြစ်သည်။ ဤ `<body`> အပိုင်းသည် HTML စာမျက်နှာ၏ အဓိက အကြောင်းအရာကို ဖော်ပြသည့် နေရာဖြစ်သည်။

**Transition Note:** အခုဆိုရင် စာမျက်နှာရဲ့ အခြေခံ Framework ကို သိသွားပါပြီ။ နောက်အပိုင်းမှာတော့ ဒီ Body ထဲမှာ ဘယ်လိုစာသားတွေနဲ့ ပုံတွေကို ထည့်မလဲဆိုတာ လေ့လာကြရအောင်။

## 3. Text & Content Structuring: စာသားနှင့် အကြောင်းအရာကို စီစဉ်ခြင်း (Structuring Text and Content)

HTML ၏ အဓိက အလုပ်မှာ အချက်အလက်များကို စနစ်တကျ စီစဉ်တင်ပြရန် ဖြစ်သည်။ စာသားများကို စနစ်တကျ စီစဉ်ခြင်းဖြင့် အကြောင်းအရာသည် ပိုမိုရှင်းလင်းပြီး နားလည်လွယ်ကူလာသည်။

### Heading Tags (ခေါင်းစဉ်များ)

အကြောင်းအရာများကို အဆင့်လိုက် ခွဲခြားတင်ပြရန်အတွက် Heading Tags များသည် အလွန်အရေးကြီးသည်။ HTML တွင် `<h1>` မှ `<h6>` အထိ ရှိသည်။ ဤ Tag များသည် အကြောင်းအရာ၏ အရေးကြီးမှုအလိုက် အဆင့်သတ်မှတ်ချက်ကို ဖော်ပြသည်။ `<h1>` သည် အကြီးမားဆုံးနှင့် အရေးအကြီးဆုံး ခေါင်းစဉ်ဖြစ်ပြီး၊ `<h6>` သည် အနည်းဆုံး အရေးကြီးသော ခေါင်းစဉ် ဖြစ်သည်။ အကြောင်းအရာကို အရေးကြီးမှုအလိုက် ခွဲခြားတင်ပြရာတွင် ဤ Hierarchy ကို အသုံးပြုခြင်းဖြင့် Search Engine များအပြင် လူသားများအတွက်ပါ အလွန်အဆင်ပြေစေသည်။

### Paragraphs နှင့် Line Breaks

စာသားများကို စီစဉ်ရာတွင် အဓိကအားဖြင့် စာပိုဒ်များ (Paragraphs) နှင့် တစ်ကြောင်းတိုးခြင်း (Line Breaks) ကို အသုံးပြုရသည်။ စာပိုဒ်တစ်ခုကို ဖန်တီးရန်အတွက် `<p>` Tag ကို အသုံးပြုသည်။ `<p>` Tag သည် စာသားအစုအဝေးတစ်ခုကို ဖော်ပြပြီး၊ စာပိုဒ်တစ်ခုပြီးတိုင်း အလိုအလျောက် အပိုကြောင်း (Margin) တစ်ကြောင်း ထည့်ပေးသည်။ တစ်ကြောင်းတိုးရန်အတွက်မူ `<br>` Tag ကို အသုံးပြုသည်။ `<br>` Tag သည် စာသားကို နောက်တစ်ကြောင်းသို့ တိုးစေရန် အသုံးပြုပြီး၊ ၎င်းသည် အကြောင်းအရာကို ခွဲခြားရန်သာ အသုံးပြုသည်။

### Basic Text Formatting (အခြေခံ စာသား ပုံစံ)

စာသားများကို ပိုမို ဆွဲဆောင်မှုရှိစေရန်အတွက် အခြေခံ ပုံစံများကို ထည့်သွင်းနိုင်သည်။ Bold (ထူသောစာသား) နှင့် Italic (စောင်းသောစာသား) တို့သည် အသုံးအများဆုံး ပုံစံများ ဖြစ်သည်။ Bold စာသားများအတွက် `<b`> သို့မဟုတ် ပိုမို အားကောင်းသော `<strong`> ကို အသုံးပြုသည်။ ထို့အတူ Italic စာသားများအတွက် `<i`> သို့မဟုတ် `<em`> ကို အသုံးပြုသည်။ ဤအရာများသည် စာသားအချို့ကို မီးမောင်းထိုးပြရန်နှင့် အရေးကြီးကြောင်း ဖော်ပြရန် အထောက်အကူပြုသည်။

**Transition Note:** စာသားတွေ စနစ်တကျ စီစဉ်ပြီးပြီဆိုတော့၊ အခုတော့ ဒီအကြောင်းအရာတွေကို ဘယ်လို ပုံစံအလှနဲ့ ပတ်သက်အောင် ပုံဖော်မလဲဆိုတဲ့ အဆင့်ကို ဆက်သွားကြရအောင်။

## 4. Attributes: Element တွေကို ပိုမို အဓိပ္ပာယ်ရှိအောင် ပြုလုပ်ခြင်း (Making Elements More Meaningful with Attributes)

HTML Element များသည် Tag များဖြင့်သာ အဓိပ္ပာယ်ပေးနိုင်သော်လည်း၊ ၎င်းတို့၏ အခြေအနေနှင့် အချက်အလက်များကို ပိုမိုတိကျစွာ ဖော်ပြရန်အတွက် Attribute များသည် မရှိမဖြစ် လိုအပ်သည်။

### Attribute ဆိုတာ ဘာလဲ? (What are Attributes?)

Attribute ဆိုသည်မှာ HTML Element များ၏ ဂုဏ်သတင်း (Properties) သို့မဟုတ် အပိုဆောင်း အချက်အလက်များကို ထည့်သွင်းရန် အသုံးပြုသည်။ Attribute များသည် Element ၏ လုပ်ဆောင်ပုံ သို့မဟုတ် ထိုElement နှင့် ဆက်စပ်နေသော အချက်အလက်များကို ပေးသည်။ Attribute များသည် Element ကို ပိုမို အဓိပ္ပာယ်ရှိအောင် ပြုလုပ်ပေးပြီး၊ Browser နှင့် အခြားကွန်ပျူတာစနစ်များအတွက် ထိုအကြောင်းအရာကို ပိုမို နားလည်စေသည်။ Attribute များကို အဖွင့် Tag အတွင်း၌ ထည့်သွင်းရပြီး၊ `name="value"` ပုံစံဖြင့် ဖော်ပြရသည်။

### အသုံးများသော Attributes များ

HTML တွင် အသုံးအများဆုံးနှင့် အရေးကြီးဆုံး Attributes များမှာ အောက်ပါအတိုင်း ဖြစ်သည်။

**`src` (Source) Attribute:** ဤ Attribute သည် ပုံများ (Images) နှင့် ဗီဒီယိုများ (Videos) ကို HTML စာမျက်နှာထဲသို့ ထည့်သွင်းရန် အသုံးပြုသည်။ ဥပမာအားဖြင့်၊ `<img src="image.jpg" alt="A description of the image">` ဟု ရေးသားသောအခါ `src` သည် ပုံဖိုင်၏ လမ်းကြောင်းကို ညွှန်ပြသည်။

**`href` (Hyperlink Reference) Attribute:** ဤ Attribute သည် Hyperlink များ (လင့်ခ်များ) ဖန်တီးရန် အသုံးပြုသည်။ ဤ Attribute ဖြင့် အခြားသော Website များ သို့မဟုတ် စာမျက်နှာများသို့ လမ်းကြောင်းပေးနိုင်သည်။ ဥပမာအားဖြင့်၊ `<a href="https://www.google.com">Go to Google</a>` ဟု ရေးသားသောအခါ `href` သည် ထိုလင့်ခ်၏ ဦးတည်ရာကို ဖော်ပြသည်။

**`alt` (Alternative Text) Attribute:** ဤ Attribute သည် ပုံများအတွက် အလွန်အရေးကြီးသည်။ ပုံကို မပြသနိုင်သည့်အခါ (သို့မဟုတ်) Screen Reader များကဲ့သို့ အထူးပံ့ပိုးမှုများအတွက် ပုံ၏ အကြောင်းအရာကို စာသားဖြင့် ဖော်ပြပေးသည်။ ၎င်းသည် Accessibility (လက်လှမ်းမီနိုင်မှု) အတွက် အလွန်အရေးကြီးသော အချက်အလက် ဖြစ်သည်။

**အနှစ်ချုပ်:** Attributes များသည် HTML Element များအတွက် အချက်အလက် (Data) ပေးခြင်း၏ အခန်းကဏ္ဍမှ ပါဝင်သည်။ ၎င်းတို့သည် Element များကို စုစည်းတင်ပြရုံမက အချက်အလက်များဖြင့် ချိတ်ဆက်ပေးနိုင်သော စွမ်းရည်ကို ပေးသည်။

**Transition Note:** Attributes တွေက Element တွေကို အချက်အလက်တွေနဲ့ ချိတ်ဆက်ပေးနိုင်ပြီ။ အခုဆိုရင် ဒီအရာအားလုံးကို ပေါင်းစပ်ပြီး ကိုယ်ပိုင် Website တစ်ခုကို စတင်တည်ဆောက်ကြည့်ရအောင်။

## 5. Practical Application: ကိုယ်ပိုင် HTML စာမျက်နှာ ရေးသားခြင်း (Creating Your Own HTML Page)

အခြေခံသဘောတရားများကို နားလည်ပြီးပြီဖြစ်သောကြောင့် လက်တွေ့တွင် HTML ကို အသုံးပြု၍ ကိုယ်ပိုင်စာမျက်နှာတစ်ခုကို ရေးသားကြည့်ကြရအောင်။ ဤအဆင့်များအတိုင်း လိုက်နာခြင်းဖြင့် သင်သည် အခြေခံဝဘ်စာမျက်နှာတစ်ခုကို တည်ဆောက်နိုင်မည် ဖြစ်သည်။

### Step 1: HTML File ကို ဖန်တီးခြင်း

ပထမဆုံးအနေဖြင့် သင်သည် Text Editor တစ်ခုကို အသုံးပြုရမည်။ ဥပမာအားဖြင့် Microsoft Notepad သို့မဟုတ် VS Code ကဲ့သို့သော Text Editor တစ်ခုကို ဖွင့်ပါ။ ထို့နောက် သင်ဖန်တီးလိုသော HTML ဖိုင်အတွက် အမည်ပေးပြီး `.html` extension ဖြင့် သိမ်းဆည်းရမည်။ ဥပမာအားဖြင့် `index.html` ဟု သိမ်းဆည်းနိုင်ပါသည်။ ဤဖိုင်သည် သင်၏ ဝဘ်စာမျက်နှာ၏ အခြေခံပုံစံ ဖြစ်သည်။

### Step 2: အခြေခံ Structure ရေးသားခြင်း

ဖိုင်ကို ဖွင့်ပြီး HTML ၏ အခြေခံဖွဲ့စည်းပုံကို စနစ်တကျ ရေးသားရမည်။ အစတွင် `<!DOCTYPE html>` ကို ရေးသားပါ။ ၎င်းသည် Browser ကို HTML5 စံနှုန်းအတိုင်း ဖတ်ရှုရန် ညွှန်ကြားခြင်း ဖြစ်သည်။ ထို့နောက် `<html`> Tag ကို အသုံးပြု၍ စာမျက်နှာတစ်ခုလုံးကို ဝန်းရံပေးရမည်။ ဤ `<html`> Tag အတွင်း၌ အကြောင်းအရာများကို ခွဲခြားရန်အတွက် `<head>` နှင့် `<body>` တို့ကို ထည့်သွင်းရမည်။ `<head>` အတွင်း၌ `<title>` ကို ထည့်သွင်းပြီး၊ `<body`> အတွင်း၌ စာမျက်နှာပေါ်တွင် မြင်တွေ့ရမည့် အကြောင်းအရာများကို ထည့်သွင်းရမည်။

### Step 3: Content ထည့်သွင်းခြင်း

အခုအခါတွင် `<body`> Tag အတွင်း၌ စာသားများကို စနစ်တကျ စီစဉ်ရမည်။ အရေးကြီးသော ခေါင်းစဉ်များအတွက် `<h1>` မှ `<h6>` Tags များကို အသုံးပြု၍ အဆင့်လိုက် ခွဲခြားတင်ပြရမည်။ စာပိုဒ်များကို ဖန်တီးရန်အတွက် `<p>` Tag ကို အသုံးပြုပြီး လိုအပ်သော စာသားများကို ထည့်သွင်းရမည်။ တစ်ကြောင်းတိုးရန် လိုအပ်သည့်နေရာများတွင် `<br>` Tag ကို အသုံးပြု၍ ခွဲခြားပေးရမည်။ ဤအဆင့်တွင် သင်သည် သင်ရေးလိုသော အကြောင်းအရာများကို HTML ၏ စည်းမျဉ်းအတိုင်း စီစဉ်တင်ပြခြင်းကို လေ့ကျင့်ရမည် ဖြစ်သည်။

### Step 4: Link နှင့် Image ထည့်သွင်းခြင်း

HTML ၏ အစွမ်းအထက်ကို အသုံးချရန်အတွက် Link နှင့် Image များကို ထည့်သွင်းရမည်။ ပုံတစ်ပုံကို ထည့်သွင်းရန်အတွက် `<img src="path/to/image.jpg" alt="Description">` ဟု ရေးသားရမည်။ ဤနေရာတွင် `src` သည် ပုံ၏ လမ်းကြောင်းကို၊ `alt` သည် ပုံ၏ အကြောင်းအရာကို ဖော်ပြသည်။ အခြား Website သို့မဟုတ် စာမျက်နှာတစ်ခုသို့ လင့်ခ်ပေးရန်အတွက် `<a href="URL">Link Text</a>` ကို အသုံးပြုရမည်။ ဤနည်းလမ်းများဖြင့် သင်သည် အကြောင်းအရာများကို အခြားအရာများနှင့် ချိတ်ဆက်ပေးနိုင်ပြီ ဖြစ်သည်။

### Step 5: ကိုယ်ပိုင် Page ကို သိမ်းဆည်းခြင်းနှင့် ကြည့်ရှုခြင်း

အဆုံးသတ်အနေဖြင့် သင်ရေးသားပြီးသော HTML ဖိုင်ကို `.html` extension ဖြင့် သိမ်းဆည်းရမည်။ ဥပမာအားဖြင့် `myfirstpage.html` ဟု သိမ်းဆည်းနိုင်သည်။ ထို့နောက် ထိုဖိုင်ကို Double Click လုပ်၍ သင်အသုံးပြုနေသော Web Browser (Chrome, Firefox စသည်) တွင် ဖွင့်ကြည့်ရမည်။ ထိုအခါ သင်ရေးသားထားသော စာသားများ၊ ပုံများနှင့် လင့်ခ်များအားလုံးကို Browser တွင် အမှန်တကယ် မြင်တွေ့နိုင်မည် ဖြစ်သည်။

## Chapter Summary & Next Steps (နိဂုံးနှင့် နောက်အဆင့်)

HTML သည် ဝဘ်စာမျက်နှာ၏ အခြေခံ အရိုးစု (Skeleton) ဖြစ်သည်။ ၎င်းသည် Element များ၊ Tag များ၊ နှင့် Attribute များဖြင့် အကြောင်းအရာများကို စနစ်တကျ တည်ဆောက်နိုင်သော ဘာသာစကား ဖြစ်သည်။ ကျွန်ုပ်တို့သည် Document Type မှစ၍ `<head>` နှင့် `<body>` ဖွဲ့စည်းပုံကို နားလည်ခဲ့ပြီး၊ ခေါင်းစဉ်များ၊ စာပိုဒ်များ၊ ပုံများနှင့် လင့်ခ်များ ထည့်သွင်းခြင်းတို့ကို လက်တွေ့လေ့ကျင့်ခဲ့ကြသည်။ Attributes များသည် Element များကို အချက်အလက်များဖြင့် ချိတ်ဆက်ပေးနိုင်သော အခန်းကဏ္ဍမှ ပါဝင်သည်။

ကျွန်ုပ်တို့သည် HTML ၏ အခြေခံများကို အောင်မြင်စွာ လေ့လာပြီးပြီ ဖြစ်သည်။ ယခုအခါတွင် ကျွန်ုပ်တို့၏ ဝဘ်စာမျက်နှာသည် အခြေခံအရိုးစုကို ရရှိခဲ့ပြီ ဖြစ်သည်။ နောက်တစ်ဆင့်အနေဖြင့် ဤအရိုးစုကို အလှအပနှင့် ဆွဲဆောင်မှုရှိသော ပုံစံအဖြစ် ပြောင်းလဲပေးမည့် CSS (Cascading Style Sheets) ကို လေ့လာရန် ပြင်ဆင်ကြရအောင်။ CSS သည် HTML ၏ အခြေခံအပေါ်တွင် အလှအပဆိုင်ရာ ဒီဇိုင်းဆွဲခြင်းကို မည်သို့ပြုလုပ်ရမည်ကို သင်ကြားပေးလိမ့်မည်။
