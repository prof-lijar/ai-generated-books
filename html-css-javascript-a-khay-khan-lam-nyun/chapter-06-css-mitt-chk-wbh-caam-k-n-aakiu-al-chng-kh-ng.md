# Chapter 6: CSS မိတ်ဆက် — ဝဘ်စာမျက်နှာကို အလှဆင်ခြင်း

HTML သည် ကျွန်ုပ်တို့၏ ဝဘ်စာမျက်နှာ၏ အ뼈 (Structure)၊ အဆောက်အအုံများ၊ နှင့် အကြောင်းအရာများကို တည်ဆောက်ပေးသည့် အခြေခံဖြစ်သည်။ ၎င်းသည် စာသားများ၊ ပုံများ၊ လင့်ခ်များနှင့် အခြားသော မီဒီယာများကို စနစ်တကျ စီစဉ်ပေးသည်။ သို့သော် ထိုအဆောက်အအုံကို လူတို့နှစ်သက်ဖွယ်ကောင်းသော အလှအပနှင့် ဆွဲဆောင်မှုရှိသော ပုံစံအဖြစ် ပြောင်းလဲပေးရန် လိုအပ်ပါသည်။ ထိုအလှဆင်ခြင်းနှင့် ပုံဖော်ပေးခြင်း၏ အခန်းကဏ္ဍကို ဖြည့်ဆည်းပေးသော နည်းပညာမှာ Cascading Style Sheets (CSS) ဖြစ်သည်။ HTML က အဆောက်အအုံ၏ ပုံစံကို ပေးသကဲ့သို့ CSS သည် ထိုအဆောက်အအုံ၏ အရောင်၊ စာလုံးပုံစံ၊ အကွာအဝေးနှင့် အပြင်အဆင်တို့ကို ထိန်းချုပ်ပေးသည်။

**Hook Concept:** HTML က ဝဘ်စာမျက်နှာရဲ့ အ뼈 (Structure) ဖြစ်တယ်ဆိုရင်၊ CSS က အဲဒီအ뼈တွေကို အလှဆင်ပြီး အသက်ဝင်လာအောင် ပုံဖော်ပေးတဲ့ ဆေးရောင်ခြယ်ပညာ (Art) လိုမျိုးပဲ ဖြစ်ပါတယ်။ ဒီအခန်းမှာ ကျွန်တော်တို့ဟာ HTML ကို အခြေခံပြီး CSS ကို အသုံးပြုကာ ဝဘ်စာမျက်နှာတွေကို ဘယ်လိုလှပအောင် ဖန်တီးနိုင်မလဲဆိုတာ လေ့လာကြပါမယ်။

## CSS ၏ အခြေခံသဘောတရားများ (The Core Concept of CSS)

### Cascading Style Sheets (CSS) ဆိုသည်မှာ ဘာလဲ?

CSS ဆိုသည်မှာ Cascading Style Sheets (အစဉ်လိုက်ပြောင်းလဲသော ပုံစံများ) ၏ အတိုကောက်ဖြစ်သည်။ ၎င်းသည် HTML (အဆောက်အအုံ၏ အ뼈) ကို အလှဆင်ရန်၊ ပုံစံသွင်းရန်နှင့် ဖော်ပြပုံ (Presentation) ကို ထိန်းချုပ်ရန် အသုံးပြုသော Style Sheet ဘာသာစကားတစ်ခုဖြစ်သည်။ CSS သည် HTML ၏ အခြေခံဖွဲ့စည်းပုံကို မပြောင်းလဲဘဲ ထိုဖွဲ့စည်းပုံပေါ်တွင် အလှအပဆိုင်ရာ စည်းမျဉ်းများကို ထည့်သွင်းပေးသည်။ ၎င်းသည် ဝဘ်စာမျက်နှာတစ်ခု၏ အမြင်အာရုံဆိုင်ရာ အစိတ်အပိုင်းများကို ထိန်းချုပ်ပေးပြီး၊ HTML ကို အလွန်ရှုပ်ထွေးသော HTML Tag များဖြင့် ရေးသားရန် မလိုအပ်ဘဲ ပိုမိုရှင်းလင်းပြီး ထိန်းချုပ်ရလွယ်ကူသော နည်းလမ်းကို ပေးသည်။

CSS ၏ အဓိက အရေးပါပုံမှာ ၎င်းသည် HTML ၏ အကြောင်းအရာ (Content) နှင့် CSS ၏ ပုံစံ (Style) ကို ခွဲခြားပေးသည်။ HTML သည် *ဘာကို* ရှိသည်ကို ဖော်ပြသည်။ CSS သည် *မည်သို့* မြင်ရမည်ကို ဖော်ပြသည်။ ဤနှစ်ခုကို ခွဲခြားခြင်းဖြင့် ကျွန်ုပ်တို့သည် ပိုမိုကောင်းမွန်သော ဒီဇိုင်းများကို ပိုမိုလွယ်ကူစွာ ဖန်တီးနိုင်သည်။

### Cascading (အစဉ်လိုက်ပြောင်းလဲခြင်း) သဘောတရား

CSS ကို အထူးခြားဆုံးနှင့် အရေးအကြီးဆုံး သဘောတရားမှာ **Cascading (အစဉ်လိုက်ပြောင်းလဲခြင်း)** ဖြစ်သည်။ ဆိုလိုသည်မှာ CSS စည်းမျဉ်းများသည် အထက်မှ အောက်သို့ အစဉ်လိုက် အကျိုးသက်ရောက်ပုံကို ဆိုလိုသည်။ ဆိုလိုသည်မှာ Style တစ်ခုသည် အခြား Style များအပေါ် မည်သို့ အကျိုးသက်ရောက်ပုံကို သတ်မှတ်ပေးသည်။

CSS သည် Style များအကြားတွင် မည်သည့် Style ကို အရင် အကျိုးသက်ရောက်စေမည်ကို စီမံပေးသည်။ ဤအစဉ်လိုက်ပြောင်းလဲခြင်း သဘောတရားသည် အလွန်အရေးကြီးသည်။ ဥပမာအားဖြင့်၊ အကယ်၍ တစ်နေရာတွင် စာလုံးအရွယ်အစားကို သတ်မှတ်ထားပြီး နောက်တစ်နေရာတွင် ထပ်မံ သတ်မှတ်ပါက၊ နောက်ဆုံးတွင် မည်သည့် Style က အနိုင်ရမည်ကို ဆုံးဖြတ်ရမည်ဖြစ်သည်။ ဤစည်းမျဉ်းများကို နားလည်ခြင်းဖြင့် ကျွန်ုပ်တို့သည် Style များအား မှန်ကန်စွာ စီစဉ်နိုင်ပြီး ဆန့်ကျင်ဘက် ဖြစ်သော Style များကြောင့် ဖြစ်ပေါ်လာမည့် ရှုပ်ထွေးမှုများကို ရှောင်ရှားနိုင်သည်။

### CSS ကို ဘယ်လိုရေးမလဲ? (Three Ways to Apply CSS)

CSS ကို HTML နှင့် ပေါင်းစပ်၍ အသုံးပြုနိုင်သော နည်းလမ်းများမှာ သုံးမျိုးရှိသည်။ ဤနည်းလမ်းများသည် Style များအား HTML နှင့် မည်သို့ ချိတ်ဆက်ရမည်ကို ဖော်ပြသည်။

**၁။ Inline Styles (တိုက်ရိုက်ပုံစံများ):**
ဤနည်းလမ်းတွင် Style ကို တိုက်ရိုက် HTML Element တစ်ခုအတွင်း၌ `style` attribute ကို အသုံးပြု၍ ရေးသားခြင်းဖြစ်သည်။ ဥပမာအားဖြင့်၊ `<p style="color: blue;">` ဟု ရေးသားခြင်းဖြစ်သည်။ ဤနည်းလမ်းသည် အလွန်ရိုးရှင်းပြီး အထူးသဖြင့် တစ်ခုတည်းသော Element ကို လျင်မြန်စွာ စမ်းသပ်ရန် သင့်လျော်သည်။ သို့သော် ဤနည်းလမ်းကို အများကြီး အသုံးပြုပါက HTML နှင့် CSS ကို ခွဲခြားရန် ခက်ခဲလာပြီး စီမံခန့်ခွဲရန် အခက်အခဲရှိလာသည်။ ထို့ကြောင့် ဤနည်းလမ်းကို အနည်းငယ်သာ အသုံးပြုသင့်သည်။

**၂။ Internal Stylesheet (အတွင်းပိုင်း ပုံစံစာရွက်):**
ဤနည်းလမ်းတွင် Style များကို HTML ဖိုင်၏ `<head>` အပိုင်းအတွင်း၌ `<style>` Tag ကို အသုံးပြု၍ ရေးသားခြင်းဖြစ်သည်။ ဤနည်းလမ်းသည် HTML ဖိုင်နှင့် CSS ဖိုင်ကို တစ်ဖိုင်တည်းတွင် ထည့်သွင်းထားသည်။ ဤနည်းလမ်းသည် HTML ဖိုင်နှင့် Style များအား အတူတူထားရှိသောကြောင့် သေးငယ်သော စာမျက်နှာများအတွက် အဆင်ပြေသည်။ သို့သော် ဝဘ်ဆိုက်ကြီးများအတွက်မူ Style များစွာကို စီမံခန့်ခွဲရန် ခက်ခဲလာသည်။

**၃။ External Stylesheet (အပြင်ပိုင်း ပုံစံစာရွက်) — အကောင်းဆုံးနည်းလမ်း:**
ဤနည်းလမ်းသည် CSS ကို သီးခြား `.css` ဖိုင်တစ်ခုအဖြစ် ရေးသားပြီး၊ ထိုဖိုင်ကို HTML ဖိုင်နှင့် သီးခြားဖိုင်အဖြစ် ထားရှိခြင်းဖြစ်သည်။ HTML ဖိုင်တွင် `<link>` Tag ကို အသုံးပြု၍ ထို CSS ဖိုင်နှင့် ချိတ်ဆက်ပေးသည်။ ဤနည်းလမ်းသည် CSS ကို စီမံခန့်ခွဲရန်အတွက် အကောင်းဆုံးနှင့် အထိရောက်ဆုံး နည်းလမ်းဖြစ်သည်။ ဤနည်းလမ်းဖြင့် Style များအား အလွယ်တကူ ပြန်လည်သုံးသပ်နိုင်ပြီး၊ ဝဘ်စာမျက်နှာများတွင် Style များအား တစ်နေရာတည်းမှ ထိန်းချုပ်နိုင်သောကြောင့် ပရော်ဖက်ရှင်နယ် အလုပ်များတွင် အသုံးပြုကြသည်။

**Transition Note:** ကျွန်ုပ်တို့သည် ဤသင်ခန်းစာတွင် **External Stylesheet** ကို အဓိကထား၍ လေ့လာသွားမည်ဖြစ်ကြောင်း အလေးပေးလိုပါသည်။ အကြောင်းမှာ External Stylesheet သည် ပရောဂျက်ကြီးများတွင် Style များအား စနစ်တကျ ထိန်းချုပ်ရန်၊ ပြန်လည်သုံးသပ်ရန်နှင့် ပရော်ဖက်ရှင်နယ် ဒီဇိုင်းများ ဖန်တီးရန်အတွက် အကောင်းဆုံးနှင့် အထိရောက်ဆုံး နည်းလမ်းဖြစ်သောကြောင့် ဖြစ်သည်။

## CSS Selector များ မိတ်ဆက်ခြင်း (Introduction to CSS Selectors)

CSS သည် မည်သည့် HTML Element ကို ပစ်မှတ်ထားပြီး Style များ ပေးမလဲဆိုသည်ကို မည်သို့သတ်မှတ်ရမည်နည်း။ ထိုအရာကို CSS Selector များက တာဝန်ယူသည်။ Selector ဆိုသည်မှာ CSS စည်းမျဉ်းများကို မည်သည့် HTML Element များအပေါ် အကျိုးသက်ရောက်စေမည်ကို သတ်မှတ်ပေးသော စာကြောင်းများဖြစ်သည်။

### Element Selector

Element Selector ဆိုသည်မှာ HTML ၏ မည်သည့် Element အားလုံးကို ပစ်မှတ်ထားမလဲဆိုတာကို သတ်မှတ်ခြင်းဖြစ်သည်။ ၎င်းသည် အလွန်ရိုးရှင်းသော Selector ဖြစ်ပြီး၊ HTML ၏ တိုက်ရိုက် Element အမျိုးအစားကို ပစ်မှတ်ထားသည်။ ဥပမာအားဖြင့်၊ `p` ဟု ရေးလိုက်လျှင် HTML တွင်ရှိသော အားလုံးသော ပုံမှန်စာကြောင်းများ (Paragraphs) အားလုံးကို ပစ်မှတ်ထားမည်ဖြစ်သည်။ သို့မဟုတ် `h1` ဟု ရေးလိုက်လျှင် HTML ရှိရှိသော ခေါင်းစဉ်အကြီးများအားလုံးကို ပစ်မှတ်ထားမည်ဖြစ်သည်။ ဤ Selector သည် အခြေခံအကျဆုံးသော ပုံစံဖြစ်သည်။

### Class Selector

Class Selector သည် အလားတူ အသွင်အပြင်ကို ပိုင်ဆိုင်သော Element များအတွက် အစုအဝေး (Group) သတ်မှတ်ရန် အသုံးပြုသည်။ ဤ Selector ကို အသုံးပြုရန်အတွက် HTML Element တစ်ခုတွင် `class` attribute ကို အသုံးပြု၍ အမည်ပေးရမည်ဖြစ်ပြီး၊ ထိုအမည်ကို CSS တွင် `.` (dot) သင်္ကေတဖြင့် စတင်၍ ရေးသားရသည်။ ဥပမာအားဖြင့်၊ `<p class="highlight">` ဟု ရေးထားပါက၊ ထိုစာကြောင်းသည် `highlight` ဟူသော Class ကို ပိုင်ဆိုင်သည်။ CSS တွင် ၎င်းကို ပစ်မှတ်ထားရန်အတွက် `.highlight { ... }` ဟု ရေးသားရမည်ဖြစ်သည်။ ဤနည်းလမ်းသည် အလားတူ Element များအား တစ်ပြိုင်နက်တည်း စုပေါင်း၍ Style ပေးရန်အတွက် အလွန်အသုံးဝင်သည်။

### ID Selector

ID Selector သည် HTML ရှိ Element တစ်ခုတည်းကို ထူးခြားစွာ ပစ်မှတ်ထားရန် အသုံးပြုသည်။ ဤ Selector ကို အသုံးပြုရန်အတွက် HTML Element တစ်ခုတွင် `id` attribute ကို အသုံးပြု၍ ထူးခြားသော အမည်ပေးရမည်ဖြစ်ပြီး၊ CSS တွင် ၎င်းကို `#` (hash) သင်္ကေတဖြင့် စတင်၍ ရေးသားရမည်ဖြစ်သည်။ ဥပမာအားဖြင့်၊ `<div id="main-header">` ဟု ရေးထားပါက၊ ထို `div` သည် `main-header` ဟူသော ID ကို ပိုင်ဆိုင်သည်။ CSS တွင် ၎င်းကို ပစ်မှတ်ထားရန်အတွက် `#main-header { ... }` ဟု ရေးသားရမည်ဖြစ်သည်။

**ID နှင့် Class တို့၏ ကွာခြားချက်:**
Class နှင့် ID တို့သည် အလွန်အရေးကြီးသော ကွာခြားချက်များရှိသည်။ **Class** သည် အလားတူ Element များအတွက် အစုအဝေး (Multiple elements) အဖြစ် အသုံးပြုသည်။ ဆိုလိုသည်မှာ ဝဘ်စာမျက်နှာပေါ်တွင် အလားတူ ပုံစံများစွာကို တစ်ပြိုင်နက်တည်း သတ်မှတ်ပေးနိုင်သည်။ **ID** သည် တစ်မူထူးခြားသော Element တစ်ခုတည်းကိုသာ ပစ်မှတ်ထားရန် အသုံးပြုသည်။ ဥပမာအားဖြင့်၊ ဝဘ်စာမျက်နှာတွင် ခေါင်းစဉ် (Header) တစ်ခုသာ ရှိမည်ဆိုလျှင် ID ကို အသုံးပြုရန် သင့်လျော်သည်။ ID များသည် CSS ၏ အခြားသော စည်းမျဉ်းများ (ဥပမာ: JavaScript) နှင့်လည်း ဆက်စပ်မှုရှိသည်။

## Style Property များဖြင့် အလှဆင်ခြင်း (Styling with Properties)

Selector များဖြင့် Element များကို ရွေးချယ်ပြီးနောက်၊ ကျွန်ုပ်တို့သည် ထို Element များအား မည်သို့ အလှဆင်မည်ကို Style Property များဖြင့် သတ်မှတ်ပေးရမည်။ Style Property များသည် Element တစ်ခု၏ အသွင်အပြင်ကို ထိန်းချုပ်ပေးသည်။

### အရောင်များ (Colors)

အရောင်များသည် CSS တွင် အခြေခံကျသော Property များထဲမှ တစ်ခုဖြစ်သည်။

*   **Text Color ပြောင်းလဲခြင်း:** စာသား၏ အရောင်ကို ပြောင်းလဲရန် `color` Property ကို အသုံးပြုသည်။ ဥပမာ၊ `color: blue;` ဟု ရေးသားပါက၊ ထိုစာသား၏ အရောင်သည် အပြာရောင် ဖြစ်သွားမည်ဖြစ်သည်။
*   **Background Color ပြောင်းလဲခြင်း:** Element ၏ နောက်ခံအရောင်ကို ပြောင်းလဲရန် `background-color` Property ကို အသုံးပြုသည်။ ဥပမာ၊ `background-color: #f0f0f0;` ဟု ရေးသားပါက၊ ထို Element ၏ နောက်ခံသည် အလွန်အဖြူရောင် (Light Gray) ဖြစ်သွားမည်ဖြစ်သည်။
*   **အရောင်များအား ကိုယ်တိုင်ရွေးချယ်ပုံ:** အရောင်များကို ရွေးချယ်ရန်အတွက် အများစုက **Hex Codes** (ဥပမာ: `#FF0000` - အနီရောင်)၊ **RGB** (Red, Green, Blue တန်ဖိုးများဖြင့် - `rgb(255, 0, 0)`) သို့မဟုတ် **Color Names** (red, blue) ကို အသုံးပြုကြသည်။ ဤနည်းလမ်းများသည် အရောင်များကို တိကျစွာ ဖော်ပြနိုင်စေသည်။

### စာလုံးပုံစံများ (Typography)

စာသားများ၏ ပုံစံကို ထိန်းချုပ်ရန်အတွက် Typography နှင့် သက်ဆိုင်သော Property များသည် အလွန်အရေးကြီးသည်။

*   **Font Family ပြောင်းလဲခြင်း:** စာသားများအတွက် အသုံးပြုမည့် စာလုံးအမျိုးအစားကို သတ်မှတ်ရန် `font-family` Property ကို အသုံးပြုသည်။ ဤ Property သည် အခြားသော Font Family များစွာကို ရွေးချယ်နိုင်ရန် ခွင့်ပြုသည်။ ဥပမာ၊ `font-family: Arial, sans-serif;` ဟု ရေးသားပါက၊ ပထမဆုံး Arial ကို ကြိုးစားမည်ဖြစ်ပြီး၊ အကယ်၍ စက်ပစ္စည်းတွင် Arial မရှိပါက နောက်ထပ် Sans-serif အမျိုးအစားများထဲမှ တစ်ခုကို အစားထိုးမည်ဖြစ်သည်။ Google Fonts ကဲ့သို့သော ပြင်ပ Font များ အသုံးပြုရန်အတွက် ၎င်းတို့ကို ဝဘ်စာမျက်နှာတွင် ထည့်သွင်းရန် လိုအပ်သည်။
*   **စာလုံးအရွယ်အစား (Size):** စာသား၏ အရွယ်အစားကို ထိန်းချုပ်ရန် `font-size` Property ကို အသုံးပြုသည်။ အရွယ်အစားကို သတ်မှတ်ရာတွင် **px** (Pixel)၊ **em** (Parent Element ၏ အရွယ်အစားအပေါ် မူတည်သော တန်ဖိုး)၊ နှင့် **rem** (Root Element ၏ အရွယ်အစားအပေါ် မူတည်သော တန်ဖိုး) တို့ကို အသုံးပြုကြသည်။ `rem` သည် Responsive Design အတွက် အလွန်အသုံးဝင်သည်။
*   **စာလုံးအလေးချိန် (Weight):** စာသား၏ အလေးချိန် (ထူထဲမှု) ကို ထိန်းချုပ်ရန် `font-weight` Property ကို အသုံးပြုသည်။ ဤ Property တွင် `normal` (ပုံမှန်)၊ `bold` (ထူ) နှင့် `100` မှ `900` အထိ တန်ဖိုးများဖြင့် သတ်မှတ်နိုင်သည်။
*   **စာလုံးပုံစံ (Style):** စာသားကို စောင်းပေးခြင်း (Italic) သို့မဟုတ် ပုံစံအမျိုးမျိုး ပြောင်းလဲခြင်းများအတွက် `font-style` Property ကို အသုံးပြုသည်။ ဥပမာ၊ `font-style: italic;` ဟု ရေးသားပါက၊ ထိုစာသားသည် စောင်းပေးထားသော ပုံစံဖြင့် ပေါ်လာမည်ဖြစ်သည်။

### Box Model အခြေခံ (Introduction to Box Model)

CSS တွင် Element တစ်ခုစီကို စတုရန်းပုံ Box တစ်ခုအဖြစ် ရှုမြင်ရန် လိုအပ်သည်။ ဤ Box Model သည် အလွန်အရေးကြီးသော အခြေခံသဘောတရားဖြစ်သည်။

*   **Box (ပတ်လည်):** HTML Element တစ်ခုလုံးသည် စတုရန်းပုံ Box တစ်ခုအဖြစ် သတ်မှတ်သည်။
*   **Content (အကြောင်းအရာ):** Box အတွင်း၌ တကယ်ပါဝင်သည့် စာသား၊ ပုံများ စသည့် အကြောင်းအရာများဖြစ်သည်။
*   **Padding (အတွင်းအနား):** Content နှင့် Box ၏ အနား (Border) ကြားရှိ အတွင်းနေရာကို Padding ဟုခေါ်သည်။ ဤနေရာသည် Content နှင့် Border ကြားတွင် အလွတ်နေရာပေးသည်။ ဥပမာ၊ `padding: 10px;` ဟု ရေးသားပါက၊ Content နှင့် Box ၏ အနားများကြားတွင် ၁၀ Pixel အတိုင်းအတာ အလွတ်နေရာ ထည့်ပေးမည်ဖြစ်သည်။
*   **Border (အနားသတ်):** Box ၏ အပြင်ဘက်တွင် ရှိသော အနားသတ်ကို Border ဟုခေါ်သည်။ ၎င်းသည် Box ကို အခြား Element များမှ ခွဲခြားပေးသည်။ `border: 1px solid black;` ဟု ရေးသားပါက၊ ၁ Pixel အနားသတ်နှင့် အမည်းရောင်မျဉ်းကြောင်းဖြင့် ပတ်လည်ဖော်ပြမည်ဖြစ်သည်။
*   **Margin (အပြင်အနား):** Box တစ်ခုနှင့် အခြား Element များကြားရှိ အပြင်ဘက် အလွတ်နေရာကို Margin ဟုခေါ်သည်။ Margin သည် Box များအကြား အကွာအဝေးကို ထိန်းချုပ်သည်။ `margin: 20px;` ဟု ရေးသားပါက၊ ထို Box နှင့် အခြား Element များကြားတွင် ၂၀ Pixel အတိုင်းအတာ အလွတ်နေရာ ထည့်ပေးမည်ဖြစ်သည်။

## လက်တွေ့ လေ့ကျင့်ခန်း (Practical Implementation)

ယခုအခါ ကျွန်ုပ်တို့သည် CSS ၏ အခြေခံသဘောတရားများကို လက်တွေ့အသုံးချမည့် အချိန်သို့ ရောက်ရှိလာပြီဖြစ်သည်။ အကောင်းဆုံးနည်းလမ်းဖြစ်သော External Stylesheet ကို အသုံးပြု၍ HTML စာမျက်နှာတစ်ခုကို အလှဆင်ကြည့်ကြပါမည်။

### လက်တွေ့ အဆင့် (Hands-on Practice)

**အဆင့် ၁: HTML စာမျက်နှာကို ပြန်လည်သုံးသပ်ခြင်း**
ပထမဆုံးအနေဖြင့် ကျွန်ုပ်တို့၏ HTML ဖိုင်ကို ပြန်လည်သုံးသပ်ရမည်။ ကျွန်ုပ်တို့သည် အဓိကအားဖြင့် `<body>` အပိုင်းနှင့် အဓိက Element များ (ဥပမာ- `<h1>`, `<p>`, `<div>`) များပေါ်တွင် အာရုံစိုက်ရမည်ဖြစ်သည်။ CSS ကို ချိတ်ဆက်ရန်အတွက် HTML ၏ `<head>` အပိုင်းတွင် External Stylesheet ကို ချိတ်ဆက်ရန် လိုအပ်သည်။

**အဆင့် ၂: External Stylesheet အသုံးပြုခြင်း**
ကျွန်ုပ်တို့သည် `styles.css` ဟု အမည်ပေးထားသော CSS ဖိုင်ကို ဖန်တီးပြီး၊ ထိုဖိုင်တွင် Style အားလုံးကို ရေးသားမည်ဖြစ်သည်။ ထို့နောက် HTML ဖိုင်၏ `<head>` အပိုင်းတွင် ဤဖိုင်ကို ချိတ်ဆက်ရန်အတွက် အောက်ပါကုဒ်ကို ထည့်သွင်းရမည်။

```html
<link rel="stylesheet" href="styles.css">
```
ဤ `<link>` Tag သည် HTML ဖိုင်နှင့် `styles.css` ဖိုင်ကို ချိတ်ဆက်ပေးသည်။ ဤနည်းလမ်းသည် Style များကို HTML နှင့် ခွဲခြားထားသောကြောင့် စီမံခန့်ခွဲရလွယ်ကူသည်။

**အဆင့် ၃: Selector များ အသုံးပြု၍ Element များအား ပစ်မှတ်ထားခြင်း**
`styles.css` ဖိုင်တွင် ကျွန်ုပ်တို့သည် မည်သည့် Element များအား Style ပေးမည်ကို Selector များဖြင့် သတ်မှတ်မည်ဖြစ်သည်။ ဥပမာအားဖြင့်၊ ကျွန်ုပ်တို့သည် အထက်တွင် ဖော်ပြခဲ့သော Class နှင့် ID Selector များကို အသုံးပြု၍ Style များ ပေးမည်ဖြစ်သည်။

```css
/* styles.css ဖိုင်အတွင်းမှ */

/* Element Selector ကို အသုံးပြုခြင်း */
body {
    font-family: 'Roboto', sans-serif; /* စာလုံးအမျိုးအစားကို ပြောင်းလဲခြင်း */
    background-color: #f4f4f9; /* နောက်ခံအရောင်ကို ပြောင်းလဲခြင်း */
    color: #333; /* စာသားအရောင်ကို ပြောင်းလဲခြင်း */
}

/* Class Selector ကို အသုံးပြုခြင်း */
.highlight {
    background-color: #fffacd; /* နောက်ခံအရောင်ကို ပြောင်းလဲခြင်း */
    border: 1px solid #ffd700; /* အနားသတ်ကို ပေးခြင်း */
    padding: 15px; /* အတွင်းအနားကို ပေးခြင်း */
    margin-bottom: 10px; /* အောက်ဘက် အပြင်အနားကို ပေးခြင်း */
}

/* ID Selector ကို အသုံးပြုခြင်း */
#main-header {
    color: #0056b3; /* ခေါင်းစဉ်စာသားအရောင် */
    font-size: 2em; /* စာလုံးအရွယ်အစားကို ကြီးစေခြင်း */
    font-weight: bold; /* စာလုံးအလေးချိန်ကို ထူစေခြင်း */
    margin-top: 20px; /* အပေါ်ဘက် အပြင်အနား */
}
```

**အဆင့် ၄: အရောင်၊ စာလုံးပုံစံနှင့် Box Model ကို အသုံးပြု၍ HTML စာမျက်နှာကို အလှဆင်ခြင်း (Practical Application)**
အထက်ပါ CSS စည်းမျဉ်းများကို HTML ဖိုင်နှင့် ပေါင်းစပ်လိုက်သောအခါ ကျွန်ုပ်တို့၏ ဝဘ်စာမျက်နှာသည် အလှဆင်မှုများဖြင့် ပြည့်စုံလာမည်ဖြစ်သည်။ ဥပမာအားဖြင့်၊ `body` အပိုင်းသည် အဖြူရောင်နှင့် နူးညံ့သော နောက်ခံရောင်ဖြင့် ပတ်လည်နေမည်။ ခေါင်းစဉ် (`#main-header`) သည် အပြာရောင်ဖြင့် ထူထဲသော စာလုံးများဖြင့် ပေါ်လာမည်။ ထို့အပြင် ကျွန်ုပ်တို့အား Class ပေးထားသော Element များ (ဥပမာ- `.highlight`) သည် ပိုမိုထူးခြားသော နောက်ခံအရောင်နှင့် အနားသတ်များဖြင့် ပေါ်လွင်လာမည်ဖြစ်သည်။ Box Model (Padding နှင့် Margin) ကို အသုံးပြုခြင်းဖြင့် အကြောင်းအရာများနှင့် အခြား Element များကြားတွင် လိုအပ်သော အလွတ်နေရာများကို စနစ်တကျ ထည့်သွင်းပေးနိုင်မည်ဖြစ်သည်။

## နိဂုံးနှင့် အနှစ်ချုပ် (Conclusion and Summary)

CSS သည် HTML ၏ အ뼈ကို အလှဆင်ပြီး အသက်ဝင်လာအောင် ပုံဖော်ပေးသော ဆေးရောင်ခြယ်ပညာပင်ဖြစ်သည်။ ကျွန်ုပ်တို့သည် CSS ၏ အဓိကသဘောတရားများဖြစ်သော Cascading သဘောတရားကို နားလည်ခဲ့ပြီး၊ Style များအား ထိန်းချုပ်ရန်အတွက် Inline၊ Internal နှင့် External Stylesheet ဟူသော နည်းလမ်းသုံးမျိုးကို လေ့လာခဲ့သည်။ အထူးသဖြင့် External Stylesheet သည် ပရော်ဖက်ရှင်နယ် ဝဘ်ဒီဇိုင်းများအတွက် အကောင်းဆုံးနှင့် အထိရောက်ဆုံး နည်းလမ်းဖြစ်ကြောင်း အလေးထားဖော်ပြခဲ့သည်။

Selector များ (Element, Class, ID) သည် HTML ၏ မည်သည့်အပိုင်းကို ပစ်မှတ်ထားမည်ကို သတ်မှတ်ပေးပြီး၊ Style Property များ (Color, Font, Box Model) သည် ထိုအပိုင်းများကို မည်သို့ ဖော်ပြမည်ကို သတ်မှတ်ပေးသည်။ Box Model (Padding, Border, Margin) သည် Element များကြားရှိ အလွတ်နေရာများနှင့် ပုံစံကို စနစ်တကျ ထိန်းချုပ်ပေးသည်။

ဤသင်ခန်းစာမှတစ်ဆင့် ကျွန်ုပ်တို့သည် ဝဘ်စာမျက်နှာကို အလှဆင်ခြင်း၏ အခြေခံကျသော အုတ်မြစ်များကို တည်ဆောက်ခဲ့ပြီဖြစ်သည်။ CSS သည် HTML နှင့် ပေါင်းစပ်၍ အလွန်စွမ်းဆောင်ရည်မြင့်မားသော၊ ဆွဲဆောင်မှုရှိသော ဝဘ်စာမျက်နှာများ ဖန်တီးနိုင်မည့် အခွင့်အလမ်းကို ပေးသည်။ နောက်အခန်းများတွင် ကျွန်ုပ်တို့သည် CSS ၏ ပိုမိုရှုပ်ထွေးသောနှင့် အဆင့်မြင့်သော အကြောင်းအရာများဖြစ်သည့် Flexbox နှင့် Grid စသည့် Layout နည်းလမ်းများကို လေ့လာသွားမည်ဖြစ်ပြီး၊ ဤအခြေခံများပေါ်တွင် အခြေခံ၍ ပိုမိုရှုပ်ထွေးသော Layout များကို ဖန်တီးနိုင်မည့် ကျွမ်းကျင်မှုများကို ရရှိလာမည် ဖြစ်သည်။

**Final Thought:** CSS သည် ကျွန်ုပ်တို့၏ ဝဘ်ဒီဇိုင်းကို ပုံသဏ္ဍာန်ပေးသည်။ အသေးစိတ်ကျကျ လေ့ကျင့်ခြင်းဖြင့် မည်သည့်ဒီဇိုင်းကိုမဆို ဖန်တီးနိုင်မည် ဖြစ်သည်။
